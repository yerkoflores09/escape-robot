# Escape Robot  
Videojuego 2D desarrollado en Python y Pygame  
Proyecto para la asignatura **Programación Orientada a Objetos – 2025**

---

## Descripción del Juego
**Escape Robot** es un videojuego tipo laberinto en 2D donde controlas a un pequeño robot que debe:

- Recorrer el mapa  
- Esquivar enemigos (fantasmas)  
- Recoger estrellas  
- Llegar al portal para avanzar al siguiente nivel  

Tu misión es escapar mientras avanzas por distintos planetas conectados por portales.

---

## Lore del Juego
Un robot viaja entre planetas y descubre un mundo con tecnología muy avanzada. Con la intención de ayudar a su propio planeta, intenta obtener información secreta, pero accidentalmente entra a una zona restringida.  

El reino de ese planeta lo descubre y comienza una persecución que lo obliga a atravesar varios mundos.  
Cada planeta está lleno de laberintos y fantasmas que intentan detenerlo.  
Si supera todos los niveles, el robot logra escapar y regresar a casa.
